Sitio web para el prendizaje de Frameworks CSS y Posicionamiento Web

El propósito del sitio web, es mostrar mediante herramientas de Frameworks CSS y Posicionamiento Web,
las librerias,diferentes estilos que a su vez permitirán ver la página mas dinámica y llamativa como resultado
de un proceso educativo digital afrontando la problemática de la pandemia del covid-19.

Frameworks CSS

Son hojas de estilos(Coding style sheets) capaces de cambiar la apariencia de una página web, como el color en una parrafo, el fondo
se puede cambiar el diseño sin alterar su contenido en HTML. Una ventaja importante es que se pueden aplicar varias hojas de estilos
en diferentes dispositivos donde se pueda visualizar la página web.

Recio García, J. A. (2016). HTML5, CSS3 y JQuery: curso práctico. RA-MA Editorial. (Pág. 146 a 185)
Recuperado de https://elibro-net.bibliotecavirtual.unad.edu.co/es/ereader/unad/106494?page=146

Posicionamiento Web

Son aquellas técnicas que se aplican al diseño para que la página pueda ser llamativa y asi mismo visible
y lider en los navegadores web, algunos elementos que se deben tener en cuenta para optimizar son:

  • Optimización de imagenes.
  • Optimización de filtros.
  • Buena estructura de las etiquetas.
  • Experiencia que tiene el usuario al navergar por la página.
    • Pedrós Piñón, M. (2018). Manual SEO: Posicionamiento web en Google para un marketing más eficaz. (Pág. 125 a 150)
