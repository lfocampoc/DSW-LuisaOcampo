Frameworks CSS

Son hojas de estilos(Coding style sheets) capaces de cambiar la apariencia de una página web, como el color en una parrafo, el fondo se puede cambiar el diseño sin alterar su contenido en HTML. Una ventaja importante es que se pueden aplicar varias hojas de estilos en diferentes dispositivos donde se pueda visualizar la página web.

Recio García, J. A. (2016). HTML5, CSS3 y JQuery: curso práctico. RA-MA Editorial. (Pág. 146 a 185)
